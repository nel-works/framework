01 / Introduction

Welcome to

NEL FW

This is an advanced battlegrounds framework with some unique features.

My framework makes developing battlegrounds genre games better and faster. It has features like code-skill builder, fast-n-safe networking system, mostly OOP based and many other features that would make the development easier.

Note: The framework is still in W.I.P. state.
02 / API Reference

API Reference

entity.new() : entity

Creates a new OOP-based entity class for Players and NPCs.

Arguments

NameTypeRequiredDescription
player(Player | Model)YesThe player entity will be based on.

Returns

A new OOP-based class self.

Example

luau 
local players        = game:GetService 'Players'
local script_service = game:GetService 'ServerScriptService'
            
local server_game  = script_service.server.game
local server_utils = script_service.server.utils

local entity_class = require( server_game.modules.entity )
local entity_list  = require( server_utils.entity_list   )

players.PlayerAdded:Connect(function(player : Player)
  local entity = entity_class.new(player)
  
  entity_list.add(entity)
  entity:initialise()

  print(entity)
end)
