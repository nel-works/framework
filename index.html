02 / Installation

Installation

Install via your package manager of choice:

bash 
# npm
npm install myproject

# yarn
yarn add myproject

# pnpm
pnpm add myproject

Requirements

  • Node.js 18+

03 / Quick Start

Quick Start

Get up and running in under a minute:

javascript 
import { MyProject } from 'myproject';

const client = new MyProject({
  apiKey: 'YOUR_API_KEY',
});

const result = await client.doSomething({ input: 'Hello world' });
console.log(result);
